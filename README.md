# 🤖 Smart Quality Inspection Agent

<p align="center">
  <img src="assets/Screenshot 2026-05-13 115140.png" alt="Smart Quality Inspection Agent Banner" width="100%">
</p>

An AI-powered quality inspection system that automates the inspection process by analyzing images, documents, and reports. The agent intelligently classifies inspection results as **Pass**, **Review**, or **Reject**, then automatically generates reports, sends notifications, creates issue tickets, and maintains centralized inspection logs.

## ✨ Features

* 🤖 AI-powered quality inspection
* 📄 Image and document analysis
* ✅ Automatic Pass, Review, and Reject classification
* 📧 Email notifications
* 📑 Inspection report generation
* 📝 Centralized logging
* ⚡ Automated workflow using n8n
